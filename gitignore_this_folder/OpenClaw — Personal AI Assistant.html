OpenClaw

The AI that actually does things.

Clears your inbox, sends emails, manages your calendar, checks you in for flights.
All from WhatsApp, Telegram, or any chat app you already use.

⟩ Quick Start

# Works everywhere. Installs everything. You're welcome. 🦞
$ powershell -c "irm https://openclaw.ai/install.ps1 | iex"
# Install OpenClaw
$ npm i -g openclaw
# Meet your lobster
$ openclaw onboard
# For those who read source code for fun
$ powershell -c "& ([scriptblock]::Create((irm https://openclaw.ai/install.ps1))) -InstallMethod git"
# Source checkouts are pnpm workspaces
$ git clone https://github.com/openclaw/openclaw.git
$ cd openclaw && corepack enable && pnpm install
# Run from source; bundled plugins use extensions/* deps
$ pnpm openclaw onboard
Companion App (Beta) Menubar access to your lobster. Works great alongside the CLI.
Download for macOS Requires macOS 15+ · Universal Binary

Works on macOS, Linux, and Windows. The one-liner installs Node.js and everything else for you. Switch later with openclaw update --channel dev or openclaw update --channel stable.
